# 11. 源码文件映射表

## 1. 启动入口

- `src/entrypoints/cli.tsx`
  最外层 CLI fast-path 分流。

- `src/main.tsx`
  主启动器与运行时装配中心。

- `src/entrypoints/init.ts`
  配置、环境、证书、遥测初始化。

- `src/entrypoints/mcp.ts`
  MCP Server 入口。

- `src/QueryEngine.ts`
  非交互 / SDK 查询生命周期核心。

## 2. REPL / UI

- `src/replLauncher.tsx`
  动态挂载 `App` 与 `REPL`。

- `src/components/App.tsx`
  顶层 Provider 容器。

- `src/screens/REPL.tsx`
  交互式会话中枢。

- `src/components/PromptInput/PromptInput.tsx`
  Prompt 输入组件核心。

- `src/hooks/useTextInput.ts`
  文本编辑、提交与快捷键。

- `src/ink/root.ts`
  Ink root 创建。

- `src/ink/components/App.tsx`
  终端输入、Raw mode、焦点处理。

- `src/ink/parse-keypress.ts`
  终端按键与鼠标事件解析。

## 3. 状态管理

- `src/bootstrap/state.ts`
  进程级全局状态。

- `src/state/AppStateStore.ts`
  AppState 结构定义。

- `src/state/store.ts`
  轻量 store 实现。

- `src/state/onChangeAppState.ts`
  AppState 副作用同步。

- `src/state/AppState.tsx`
  React 侧 AppState Provider 与 hooks。

## 4. 查询执行

- `src/query.ts`
  agent turn 主循环。

- `src/query/deps.ts`
  query 的依赖注入层。

- `src/query/stopHooks.ts`
  stop hooks 处理。

- `src/services/tools/toolOrchestration.ts`
  工具批次编排。

- `src/services/tools/StreamingToolExecutor.ts`
  流式工具执行路径。

- `src/utils/handlePromptSubmit.ts`
  Prompt 提交统一入口。

- `src/utils/processUserInput/processUserInput.ts`
  query 前用户输入处理。

## 5. 工具系统

- `src/Tool.ts`
  工具契约与 `ToolUseContext`。

- `src/tools.ts`
  工具池装配。

- `src/tools/BashTool/*`
  shell 工具与权限规则。

- `src/tools/FileReadTool/*`
  文件读取。

- `src/tools/FileEditTool/*`
  文件编辑。

- `src/tools/SkillTool/*`
  skill 执行入口。

- `src/tools/AgentTool/*`
  子代理能力。

- `src/tasks/*`
  task / teammate / workflow 运行时实体。

- `src/coordinator/coordinatorMode.ts`
  coordinator mode 切换与工具过滤边界。

- `src/tools/ToolSearchTool/*`
  deferred tools 检索与选择。

## 6. 命令 / Skills / 插件

- `src/commands.ts`
  命令聚合入口。

- `src/types/command.ts`
  Command 契约定义。

- `src/skills/loadSkillsDir.ts`
  本地 skill 加载器。

- `src/plugins/builtinPlugins.ts`
  built-in plugin 注册。

- `src/utils/plugins/loadPluginCommands.ts`
  plugin commands / skills 装配入口。

- `src/utils/plugins/pluginLoader.ts`
  plugin 发现、校验、加载、合并。

## 7. MCP

- `src/services/mcp/client.ts`
  MCP Client 核心实现。

- `src/services/mcp/config.ts`
  MCP 配置解析。

- `src/services/mcp/auth.ts`
  MCP OAuth / auth 流程。

- `src/services/mcp/types.ts`
  MCP 配置、连接、客户端状态类型。

- `src/services/mcp/useManageMCPConnections.ts`
  MCP 连接接入与状态更新。

- `src/services/mcp/MCPConnectionManager.tsx`
  UI 层连接管理。

## 8. remote / bridge / direct connect / viewer

- `src/remote/RemoteSessionManager.ts`
  remote session 管理。

- `src/remote/SessionsWebSocket.ts`
  remote session WebSocket 订阅。

- `src/remote/sdkMessageAdapter.ts`
  SDK message 到本地消息结构的适配。

- `src/hooks/useRemoteSession.ts`
  remote mode React 入口。

- `src/bridge/initReplBridge.ts`
  bridge 初始化入口。

- `src/bridge/replBridge.ts`
  bridge 会话写入与同步。

- `src/bridge/remoteBridgeCore.ts`
  bridge 远程通信核心。

- `src/hooks/useReplBridge.tsx`
  bridge mode React 入口。

- `src/hooks/useSSHSession.ts`
  `claude ssh` 在 REPL 侧的远程会话接入。

- `src/server/createDirectConnectSession.ts`
  direct connect 会话创建。

- `src/server/directConnectManager.ts`
  direct connect 会话管理。

- `src/hooks/useDirectConnect.ts`
  direct connect React 入口。

- `src/hooks/useAssistantHistory.ts`
  assistant viewer 历史分页入口。

- `src/assistant/sessionHistory.ts`
  assistant viewer 历史拉取实现。
